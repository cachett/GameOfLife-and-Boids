Les scripts pour ce projet sont separes en trois parties (repertoire partie2, partie3, partie4).
Un Makefile se trouve dans chacun d'entre eux.
Pour compiler, il suffit de se rendre dans le repertoire en question et de taper en ligne de commande:

make all

ou

pour partie 2:
make TestBallsSimulator

pour partie 3:
make TestCellsSimulator
make TestCellsNStatesSimulator
make TestHabitationsSimulator

pour partie 4
make BoidsSimulator
make BoidsMultiSimulator

Enfin pour executer le programme:

java nom_du_programme
